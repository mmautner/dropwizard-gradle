# dropwizard-gradle

Simple Dropwizard web app with Gradle:

```
$ ./gradlew shadowJar
$ java -jar build/libs/dropwizard-example-1.0-SNAPSHOT-all.jar
```

Served at `http://localhost:8080/hello`
